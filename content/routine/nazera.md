# নাযেরা বিভাগের ক্লাসরুটিন

| সময় | বিষয় |
|------|-------|
| বাদ ফযর | সূরা ইয়াসিনের আমল ও কুরআন তেলাওয়াত |
| ফযরের ৪০ মিনিট পর | হালকা নাস্তা (১৫ মিনিট) |
| অতঃপর ৮:৩০ মিনিট পর্যন্ত | কুরআন শরীফ ও তাজভীদুল কুরআন |
| ৮:৩০ মিনিট হতে ৯:০০ পর্যন্ত | নাস্তা |
| ৯:০০ টা হতে ১১:০০ পর্যন্ত | ঘুম |
| ১১:০০ হতে ১১:৩০ পর্যন্ত | গোসল |
| ১১:৩০ হতে যোহরের আযান পর্যন্ত | বাংলা, গণিত ও ইংরেজী |
| যোহরের পর হতে ২:৩০ পর্যন্ত | নামায ও খানা |
| ২:৩০ মিনিট হতে আছরের আযানের ৪০ মিনিট পূর্ব পর্যন্ত | কুরআন শরীফ |
| অতঃপর ৪০ মিনিট | মাসনূন দো'আ ও হাদীস |
| বাদ আছর হতে মাগরিবের আযানের ২০ মিনিট পূর্ব পর্যন্ত | বিরতি / হাতের লেখা / শরীরচর্চা |
| বাদ মাগরিব হতে ইশার আযানের ১০ মিনিট পূর্ব পর্যন্ত | কুরআন শরীফ, বাংলা ও ইংরেজী পড়া (ইয়াদ) |
| ইশার নামাযের ১০ মিনিট পূর্ব পর্যন্ত | রাত্রির খানা |
| ইশার নামাযের পর থেকে রাত ১০:০০ পর্যন্ত | গণিত |
| রাত ১০:০০ টার পর | ঘুম |

## বি. দ্র.
- অভিভাবকগণ প্রতিদিন **বাদ আছর থেকে মাগরিবের ২০ মিনিট পূর্ব পর্যন্ত**, এবং **শুক্রবার সকাল ৮টা থেকে ১০টা পর্যন্ত** সাক্ষাৎ করতে পারবেন।  
- নাযেরা (মক্তব ২য় বর্ষ) ছাত্ররা **প্রতি ১ মাস পরপর** বৃহস্পতিবার সকাল ৯টা থেকে শুক্রবার আসর পর্যন্ত বাসায় যেতে পারবে। তবে **শুক্রবার মাগরিবের পূর্বে অবশ্যই ফিরে আসতে হবে**।  
- উল্লিখিত রুটিন অবশ্যই যথাযথভাবে মানতে হবে। অমান্য করলে উপযুক্ত ব্যবস্থা নেওয়া হবে।  
- মৌসুমভেদে সময়সূচী পরিবর্তন হতে পারে।
